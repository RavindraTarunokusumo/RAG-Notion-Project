# src/tools/math_solver.py

## Purpose
Tool agent for symbolic math operations using SymPy.

## Main responsibilities
- Advertise math-solving capabilities.
- Parse and evaluate/simplify/solve/differentiate/integrate expressions.
- Return string and LaTeX output representations.

## Key symbol
- `MathSolverAgent`

## Supported operations
- `simplify`, `solve`, `diff`, `integrate`, `evaluate`
