# Product Definition

## Initial Concept
A multi-agent RAG system that answers research questions using your Notion knowledge base and arXiv papers. Features dynamic tool agents, real-time streaming, and [LangSmith](https://docs.langchain.com/langsmith/home) tracing.

## Target Audience
- Students

## Core Capabilities
- Provide cited answers to research questions using Notion and arXiv.
- Visualize concepts with dynamically generated diagrams.

## Problem Statement
- Difficulty in synthesizing information from scattered Notion notes and academic papers.

## Key Features
- Multi-agent collaboration (Planner, Researcher, Reasoner, Synthesiser).
- Real-time streaming and dynamic tool usage.
- Integration with Notion API and arXiv for data sources.

## Value Proposition
- A single interface to query both personal knowledge and academic literature.
